# Week 1 - JavaScript Fundamentals Assignment

## Overview
This folder contains JavaScript exercises focused on fundamental programming concepts including conditional statements, loops, functions, and array operations. The assignments are organized by day and progressively build upon core JavaScript skills.
##  Folder Structure

### Day 1 - Array Operations & Conditionals
Exercises focused on working with numbers and arrays using basic control flow.

#### Files:

1. 2largest.js
   - Finds the largest of 2 numbers
   - Uses if-else conditional statements
   - Demonstrates basic comparison operators

2. 3largest.js
   - Finds the largest of 3 numbers
   - Uses if-else conditions with multiple comparisons
   - Introduces nested conditional logic

3. function1.js
   - Function-based approach to finding the largest of 3 numbers
   - Demonstrates function definition and return values
   - Encapsulates comparison logic in reusable code

4. function2.js
   - Function to calculate the sum of array elements
   - Iterates through arrays using for loops
   - Introduces accumulator pattern

5. function3.js
   - Function to search for an element in an array
   - Returns the index of the found element
   - Demonstrates array searching logic

6. smallinarray.js
   - Finds the smallest/minimum number in an array
   - Uses comparison logic in a loop
   - Introduces min-value pattern

7. sumarray.js
   - Calculates the total sum of array elements
   - Uses template literals for output formatting
   - Demonstrates array iteration techniques

### Day 2 - Advanced Array Operations
Introduction to complex data structures and array manipulation methods.

#### Files:

1. AdvArrayAssign.js
   - Works with array of objects (employee data structure)
   - Demonstrates array methods: `splice()`, `push()`, `pop()`
   - Includes three operations:
     - Insert: Add new employee at specific position
     - Delete: Remove employee by name
     - Update: Modify nested array values
   - Introduces object-oriented data handling

## Key Concepts Covered

- Conditionals: if, else if, else statements
- Operators: Comparison operators (>, <, ==, ===)
- Loops: for loops for array iteration
- Functions: Function definition, parameters, return values
- Arrays: Array indexing, iteration, manipulation
- Objects: Object literals, nested structures
- Array Methods: splice(), push(), pop()
- Variables: Variable declaration and assignment
